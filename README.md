# DevPars — Software & Web (100% remote)

Trilingual (DE / EN / FA) freelance portfolio & services site.
Static HTML/CSS/JS, no build step. Self-hosted fonts (DSGVO-safe), no cookies, no tracking.

- `index.html` — main page (services, work, pricing, process, about, contact)
- `impressum.html` / `datenschutz.html` — legal pages (DE)
- `assets/` — css, js, self-hosted woff2 fonts, SVG mockups
- `card/` — shareable business card for Telegram / messengers

Deployed on GitHub Pages.
